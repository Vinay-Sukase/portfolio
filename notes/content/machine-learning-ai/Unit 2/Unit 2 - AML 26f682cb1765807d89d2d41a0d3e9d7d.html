Regression

Regression is a supervised machine learning technique used for predicting continuous numerical values. It models the relationship between dependent and independent variables using a mathematical function.

Regression is used to :

  • predicting house price from area, location, rooms.
  • predicting student marks from hours studied.
  • predicting sales based on ad spending.
  • predicting temperature tomorrow based on past days.

Gradient Descent

Gradient descent is a mathematical technique that iteratively finds the weights and bias that produce the model with the lowest loss. Gradient descent finds the best weight and bias by repeating the following process for a number of user-defined iterations.

The model begins training with randomized weights and biases near zero, and then repeats the following steps:

  1. Calculate the loss with the current weight and bias.
  1. Determine the direction to move the weights and bias that reduce loss.
  1. Move the weight and bias values a small amount in the direction that reduces loss.
  1. Return to step one and repeat the process until the model can't reduce the loss any further.

Assumptions for Multiple Linear Regression

To produce reliable results, MLR relies on the following assumptions:

  1. Linearity: Relationship between dependent and independent variables is linear.
  1. No Multicollinearity: Independent variables should not be highly correlated with each other.
  1. Homoscedasticity: Constant variance of residuals.
  1. Normality: Errors should be normally distributed.
  1. Independence of Errors: No autocorrelation in error terms.

Violating these assumptions can lead to misleading or unstable predictions.

Logistic Regression

Logistic Regression is a supervised learning algorithm used for classification problems, where the output variable is categorical (usually binary, e.g., 0/1, Yes/No, True/False). Unlike Linear Regression, which predicts continuous values, Logistic Regression predicts the probability of an event occurring and maps it into the range [0,1] using the sigmoid (logistic) function.

For example:

Support Vector Machines

Support Vector Machine is a supervised learning algorithm used for classification and regression. It finds the optimal hyperplane that separated data points of different classes with the maximum margin. SVM is effective for both linear and nonlinear classification problem using kernel functions.

Kernel Functions

A Kernel Function is a mathematical tool used in Support Vector Machines (SVMs) and other algorithms to compute similarity between data points in a high-dimensional feature space without explicitly transforming the data. This allows the algorithm to handle nonlinear relationships efficiently.

Motivation

  • Many real-world datasets are not linearly separable in their original feature space.
  • Kernel functions map the original input space to a higher-dimensional space where the data becomes linearly separable.
  • This is called the “kernel trick”, which avoids the computational cost of explicitly performing the high-dimensional transformation.

Mathematical Definition

A Kernel Function K(xi,xj)K(x_i,x_j)K(xi​,xj​)﻿ computes the inner product of two vectors xix_ixi​﻿ and xjx_jxj​﻿ in the transformed feature space ϕ(x):\phi (x):ϕ(x):﻿

Where:

  • ϕ(x)\phi(x)ϕ(x)﻿ = mapping function from original space to higher-dimensional space
  • xi,xjx_i,x_jxi​,xj​﻿ = input vectors

Key Idea:

We never need to calculate ϕ(x)\phi(x)ϕ(x)﻿ explicitly; the kernel function directly gives the inner product in the high-dimensional space.

Advantages of Kernel Functions

  1. Handle nonlinear relationships without explicitly computing high-dimensional mapping
  1. Efficient computationally using the kernel trick
  1. Flexible: choice of kernel can adapt to different data structures

Classification

Classification is a supervised machine learning technique used for predicting categorical (discrete) class labels.

It models the relationship between input features (independent variables) and a class label (dependent variable) and assigns each input to one of the predefined classes.

Classification is used to:

  • predicting whether an email is spam or not spam.
  • predicting pass or fail of a student based on marks and attendance.
  • predicting disease present or not present from medical test data.
  • predicting customer churn (yes / no) in telecom services.
  • predicting handwritten digit (0–9) from image pixels.

K-Nearest Neighbors (KNN)

K-Nearest Neighbors (KNN) is a supervised learning algorithm used for classification and regression.

It classifies a new data point based on the majority class of its k nearest neighbors in the feature space.

KNN is called a lazy learning algorithm because it does no training beforehand and makes decisions only at prediction time.

Key Concepts

  • k: Number of nearest neighbors considered.
  • Distance Metric: Usually Euclidean distance.
  • Neighbors: The k closest data points to the test point.
  • Decision Rule:
    • Classification: Majority voting
    • Regression: Average of neighbors’ values

KNN Algorithm Steps

  1. Choose value of k

    Example: k = 3

  1. Calculate distance
    • Compute distance between the test point and all training points
  1. Find k nearest neighbors
    • Select k points with minimum distance
  1. Voting / Averaging
    • Classification → Majority class
    • Regression → Mean of values
  1. Assign output
    • Final class/value is assigned to the test point
